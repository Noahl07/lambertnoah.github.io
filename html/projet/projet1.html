Présentation du BTS SIO


🧱 Projets réalisés au lycée (exemple à compléter avec les tiens) Projet 1 – Application de gestion de tâches (ToDoList) Contexte : Projet scolaire réalisé en groupe dans le cadre du module de développement web. Objectifs : Créer une application web permettant de gérer des listes de tâches (ajout, modification, suppression). Travail réalisé : Conception de la base de données MySQL, développement de l’interface en PHP et JavaScript, gestion de l’authentification. Outils et technologies : HTML, CSS, PHP, MySQL, Bootstrap, Visual Studio Code. Compétences mobilisées : C3, C4, C5. Résultats obtenus : Application fonctionnelle, interface ergonomique, hébergement local via XAMPP. Bilan personnel : Bonne mise en pratique des notions de base du développement web et du travail en équipe.
