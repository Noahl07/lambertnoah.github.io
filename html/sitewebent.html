Projet 1 — Site web d’entreprise (stage)

Conception et développement d’un site web “de A à Z” pour une entreprise, dans le cadre de plusieurs périodes de stage.

Contexte : Stage en entreprise
Durées : 2 stages (7 semaines) + prochain stage début janvier (5 semaines)
Objectif : Site vitrine + fonctionnalités métier + base de données
État : En cours

Missions réalisées

  • Recueil des besoins et structuration des contenus (informations de l’entreprise, services, contacts).
  • Intégration de données fournies par l’entreprise (textes, visuels, éléments techniques).
  • Conception de l’architecture du site (navigation, pages, parcours utilisateur).
  • Création d’une maquette pour une fonctionnalité métier : création de pièces industrielles (workflow / écrans).
  • Mise en place d’une base technique évolutive (structure du code, organisation des fichiers, styles).

Fonctionnalités prévues

  • Authentification (système de login) : à démarrer.
  • Espace de vente (fournitures / catalogue) : en cours, non finalisé.
  • Module métier : saisie / création de pièces industrielles (à partir de la maquette).
  • Gestion des contenus : pages administrables et évolution des informations.
  • Formulaire de contact et demande de devis.

Compétences mobilisées

  • HTML / CSS / JavaScript : intégration et interactions.
  • Structuration et qualité du code (réutilisabilité, maintenance).
  • Base de données : préparation des entités et des données à intégrer.
  • Sécurité web : principes pour le futur login (sessions, validation côté serveur).
  • Travail en entreprise : organisation, itérations, retours utilisateur.

Suite prévue (stage de janvier — 5 semaines)

  • Implémentation du système de login + gestion des rôles (utilisateur / admin).
  • Finalisation du module “vente de fournitures” (produits, panier, commandes).
  • Connexion complète au back-end et à la base de données.
  • Amélioration UX/UI à partir des retours (responsive, cohérence visuelle, accessibilité).
  • Tests et sécurisation (contrôles, erreurs, validation des formulaires).
