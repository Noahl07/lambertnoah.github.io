Présentation du BTS SIO


Thématique : L’intelligence artificielle générative dans le développement web État de l’art initial L’intelligence artificielle générative regroupe les technologies capables de créer du contenu automatiquement (texte, image, code, etc.) à partir de modèles entraînés. Dans le développement web, elle permet d’accélérer la création d’interfaces, la génération de code et l’assistance au débogage.
