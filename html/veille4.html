GitHub Copilot (VS Code) : agents, sessions et intégrations améliorées

Date : 28 octobre 2025
Thème : IA & productivité développeur
Contexte : GitHub Universe / VS Code

Contexte

Les assistants IA évoluent vers des fonctionnalités plus “agentiques” : ils ne se limitent plus à compléter du code, mais aident à planifier, exécuter et suivre des tâches de développement à travers des “sessions” et des workflows plus complets.

Résumé

GitHub Copilot dans Visual Studio Code a été renforcé avec des améliorations autour des agents et des sessions (Agent Sessions), et une intégration plus native dans l’éditeur. L’objectif est de faciliter le suivi des tâches et d’améliorer la continuité entre l’IDE, les outils Copilot et les flux de travail (planification, exécution, revue).

Points clés

  • Améliorations des “Agent Sessions” pour suivre/relancer les tâches.
  • Intégration plus native du “coding agent” dans l’expérience VS Code.
  • Objectif : orchestrer plus facilement les tâches IA dans le workflow dev.
